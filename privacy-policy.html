개인정보처리방침

최종 수정일: 2026년 4월 30일

1. 수집하는 개인정보

하루 앱은 개인정보를 수집하지 않습니다.

  • 일기, 트래커 기록, 사진 등 모든 데이터는 사용자의 기기와 iCloud 계정에만 저장됩니다.
  • 외부 서버로 데이터를 전송하지 않습니다.
  • 계정 생성이 필요하지 않습니다.

2. 데이터 저장 위치

데이터저장 위치
일기 내용, 사진기기 로컬 + iCloud (선택)
트래커 기록기기 로컬 + iCloud (선택)
앱 설정기기 로컬
패스코드기기 키체인 (SHA-256 해시)

3. iCloud 동기화

  • iCloud 동기화는 Apple의 CloudKit을 통해 이루어집니다.
  • 데이터는 사용자의 Apple ID에 연결된 iCloud 계정에 저장됩니다.
  • 개발자는 사용자의 iCloud 데이터에 접근할 수 없습니다.

4. 분석 데이터

  • Firebase Analytics, Crashlytics, Performance Monitoring(iOS 전용)을 통해 익명 사용 통계를 수집할 수 있습니다.
  • 수집 항목:
    • 앱 실행/일기 작성/트래커 토글 등 35종 사용 이벤트 (이벤트명만, 일기 내용은 절대 미포함)
    • 크래시 로그 + 비치명적 에러 + breadcrumb (스택 트레이스, 앱 버전, OS 버전)
    • 화면 진입 시간, 앱 시작 시간, 네트워크 호출 지연 (iOS만)
    • 사용자 프로퍼티(익명): 일기 수, 연속 기록, 테마, OS 플랫폼 등
  • 익명 사용자 식별자(User ID): 첫 실행 시 무작위 UUID를 생성해 같은 사용자의 여러 세션을 묶어 분석합니다. 이름/이메일/전화번호 등 실명 정보와 연결되지 않습니다.
  • 개인 식별 정보(일기 내용, 사진, 위치, 이름, 이메일 등)는 일체 수집하지 않습니다.
  • 광고 식별자(IDFA)는 사용하지 않으며, 다른 앱/웹사이트에서의 활동 추적도 하지 않습니다.
  • 설정 → 개인정보 → "분석 데이터 수집" 토글로 거부할 수 있으며, 거부 시 Firebase 초기화 자체가 일어나지 않아 즉시 모든 수집이 중단됩니다.

5. 제3자 제공

하루 앱은 개인정보를 제3자에게 제공하지 않습니다.

6. 데이터 삭제

  • 앱을 삭제하면 기기에 저장된 모든 데이터가 삭제됩니다.
  • iCloud 데이터는 설정 > Apple ID > iCloud에서 직접 삭제할 수 있습니다.
  • 앱 내에서 일기를 삭제하면 30일 후 영구 삭제됩니다.

7. 아동 보호

하루 앱은 만 14세 미만 아동의 개인정보를 의도적으로 수집하지 않습니다.

8. 변경 사항

본 개인정보처리방침이 변경될 경우, 앱 업데이트 또는 이 페이지를 통해 안내합니다.

9. 문의

Privacy Policy

Last updated: April 30, 2026

1. Data We Collect

Haru does not collect personal information.

  • All data (journal entries, tracker records, photos) is stored only on your device and your iCloud account.
  • No data is transmitted to external servers.
  • No account creation is required.

2. Data Storage

DataLocation
Journal entries, photosDevice + iCloud (optional)
Tracker recordsDevice + iCloud (optional)
App settingsDevice only
PasscodeDevice Keychain (SHA-256 hashed)

3. iCloud Sync

  • iCloud sync uses Apple's CloudKit.
  • Data is stored in your Apple ID-linked iCloud account.
  • The developer cannot access your iCloud data.

4. Analytics

  • Anonymous usage statistics may be collected via Firebase Analytics, Crashlytics, and Performance Monitoring (iOS only).
  • Collected:
    • 35 product interaction events (event names only — never journal content)
    • Crash logs, non-fatal errors, breadcrumbs (stack trace, app/OS version)
    • Screen render times, app start time, network latency (iOS only)
    • Anonymous user properties: entry count, current streak, theme, OS platform
  • Anonymous user ID: A random UUID is generated on first launch to link sessions of the same user for analysis. It is not linked to your real identity (name/email/phone).
  • No personally identifiable information (journal content, photos, location, names, emails) is collected.
  • IDFA (advertising identifier) is not used. No cross-app/website tracking.
  • You can opt out in Settings → Privacy → "Analytics Data Collection". When disabled, Firebase initialization itself is skipped and all collection stops immediately.

5. Third-Party Sharing

Haru does not share personal data with third parties.

6. Data Deletion

  • Uninstalling the app deletes all local data.
  • iCloud data can be deleted via Settings > Apple ID > iCloud.
  • Deleted journal entries are permanently removed after 30 days.

7. Children's Privacy

Haru does not knowingly collect data from children under 14.

8. Changes

Changes to this policy will be communicated through app updates or this page.

9. Contact

Privacy inquiries: twoface4@yahoo.co.kr
