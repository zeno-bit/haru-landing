네 가지 약속

하루가 사용자에게 진심을 담아 제공하는 것들.

📅

날짜에 묶이는 모든 것

가벼운 행동 기록부터 다가오는 디데이까지 — 일자와 관련된 다양한 기록을 한곳에 남기고, 캘린더와 타임라인에서 편하게 다시 볼 수 있어요.

📈

쌓일수록 보이는 패턴

운동 기록이든 여행이든, 꾸준히 쌓아둔 데이터는 연속 기록 마일스톤 · 트래커 주기 분석 · 월간 페이스 · 기록의 무게 같은 통계로 인사이트를 보여줘요.

🔐

안전하고 자유로운 데이터

다른 일기 앱에서 쓰던 글을 그대로 가져오고, 여러 형식으로 내보낼 수 있어요. 일기는 패스코드와 Face ID로 잠그고 사용자 본인의 iCloud에만 저장돼요.

✍️

가볍게도, 풍부하게도

"오늘 비 옴" 한 줄 메모처럼 적어도 되고, 마크다운 · 음성 입력 · 사진 첨부로 풍부하게 남겨도 돼요. 그날의 기분과 손에 잡히는 만큼만.

당신의 일기는 당신만의 것

하루는 일기 · 사진 · 위치 · 트래커 데이터를 외부 서버로 보내지 않습니다. 모든 데이터는 사용자의 기기와 사용자 본인의 iCloud 계정에만 저장됩니다.

계정 가입이 필요 없고, 광고가 없고, 결제가 없고, 광고 추적자(IDFA)를 사용하지 않습니다.

데이터 저장 기기 + 사용자 iCloud(CloudKit)만
계정 가입/로그인 불필요
광고 없음
크로스앱 추적 없음 (IDFA 미사용)
분석 데이터 옵트인 · 익명 · 거부 가능
패스코드 키체인에 SHA-256 해시로만
